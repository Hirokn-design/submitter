Submitter 📱 (v 1.00)
Submitter is a native Android application developed as part of my Computer Statistics studies at ASE. It is designed to collect user information through an optimized and intuitive form.

🚀 Features
Complete Form: Fields for Last Name, First Name, Gender, Date of Birth, Phone, and Email.

Responsive Interface: Built using ScrollView and LinearLayout to ensure compatibility across all screen sizes.

Circular Profile Picture: Integrated circular ImageView with dual-source support:

Choose an image from the Gallery.

Take a live photo using the Camera.

Material Design: Custom buttons with backgroundTint and rounded corners for a modern User Experience (UX) (Work in progress).

🛠️ Technologies
Language: Java

IDE: Android Studio

Units: DP/SP for multi-screen compatibility.

Version Control: Git & GitHub
